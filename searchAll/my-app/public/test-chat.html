🧪 تست ویجت چت

ویجت چت درون‌برنامه‌ای با قابلیت چت زنده

کنترل‌های تست

ویژگی‌های ویجت

  • طراحی مدرن و زیبا با انیمیشن‌های نرم
  • چت زنده با Socket.IO
  • نمایش وضعیت اتصال (آنلاین/آفلاین)
  • نشانگر تایپ کردن
  • اعلان‌های پیام جدید
  • ذخیره پیام‌ها در localStorage
  • پاسخ‌های خودکار برای سوالات رایج
  • طراحی واکنش‌گرا برای موبایل
  • پشتیبانی از حالت تاریک
  • امنیت XSS و محافظت از ورودی‌ها

نحوه استفاده

برای کاربران:
  • روی دکمه چت کلیک کنید
  • پیام خود را بنویسید
  • Enter یا دکمه ارسال را بزنید
  • پاسخ‌های خودکار دریافت کنید
برای توسعه‌دهندگان:
  • window.chatWidget.open()
  • window.chatWidget.close()
  • window.chatWidget.send('پیام')
  • کاملاً قابل شخصی‌سازی

کد نمونه

// باز کردن چت
window.chatWidget.open();

// بستن چت
window.chatWidget.close();

// ارسال پیام
window.chatWidget.send('سلام');

// اضافه کردن به صفحه
<link rel="stylesheet" href="chat-widget.css">
<script src="/socket.io/socket.io.js"></script>
<script src="chat-widget.js"></script>
